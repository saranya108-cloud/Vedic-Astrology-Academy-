# AI Projects Command Center

A clean, Markdown-only packet for tracking AI projects — readable in **Obsidian**, on **GitHub**, and by **AI coding assistants**.

## What this is

- 📋 A lightweight dashboard for active projects, ideas, and progress notes
- 🤝 A context packet you can hand to an AI assistant so it knows where things stand
- 📱 Simple enough to read and update from an iPhone (GitHub app or Obsidian)

## What this is NOT

- ❌ Not a full Obsidian vault — only selected, shareable notes live here
- ❌ Not a place for private personal, medical, financial, or account information
- 🔒 This is a private working repo

## How to use it

1. Open [AI_PROJECTS_DASHBOARD.md](AI_PROJECTS_DASHBOARD.md) to see project status at a glance.
2. Start a new project by copying [AI_PROJECT_TEMPLATE.md](AI_PROJECT_TEMPLATE.md).
3. When handing work to an AI assistant, point it at [FABEL_CONTEXT.md](FABEL_CONTEXT.md) first.
4. Add a dated line to [PROJECT_LOG.md](PROJECT_LOG.md) whenever something meaningful changes.

Keep edits small. Keep notes short. The dashboard works because it stays simple.
